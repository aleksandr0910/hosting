Github pages

Github pages er en statisk hosting service som tar et github repository og hoster det på et eget domain. Nettsiden er tilgjengelig for alle.

Github pages


Github pages er simpelt og har enkel dokumentasjon på hvordan man setter det opp

Det er 100% gratis


Lar deg ikke bruke frameworks som ejs



Step by step


Step 1

Lag et github repository

NB SELVOM DOKUMENTASJONEN SIER MOTSATT TRENGER DU IKKE Å KALLE DET FOR BRUKERNAVNET DITT


Step 2

Klon repositorien din inn i kodeprogrammet ditt med


git clone "https linken fra github"


Step 3

Gå på github til repository'et ditt og under settings


På settings gå til pages på menyen


Der det står branch velg main branchen istedet for none


Og nettsiden din er klar på denne lenken
